
# FreePlay Telegram Platform

## Features
- Telegram Bot (Free-to-play games)
- Points / XP system
- Anti-spam cooldown
- Admin Web Dashboard
- Railway-ready

## Env Vars
- TELEGRAM_TOKEN
- BACKEND_URL
- PORT=8080

## Deploy
1. Push to GitHub
2. Connect Railway
3. Set env vars
4. Deploy
5. Run bot
